航班名单转换工具

上传系统导出的航班名单 Excel，自动转换为海关上传模板，并生成可预览、可下载的结果表。全程在本地浏览器处理，数据不上传服务器。

✈️
📋 转换规则说明 点击展开查看
▼
系统按表头名称自动识别各列，将其清洗、映射为标准海关上传格式。下方为关键处理规则：
  • 列识别：自动匹配 航班日期 / 实际承运人 / 实际航班号 / 起飞地 / 目的地 / 名称 / 性别 / 生日 / 国籍 / 证件号码 等列。
  • 国籍代码：二字码自动转为三字码（如 CN→CHN、US→USA），并修正常见错别字；无法识别时标黄提示手动核查。
  • 性别代码：1→M、2→F（也直接接受 M/F）。
  • 证件号码：去除开头多余符号（- / ,），按 / 取第一段。
  • 出生日期：统一清洗为 YYYY-MM-DD。
  • 出入境判定：按航班号判定——MU261 / MU751 为出境，MU262 / MU752 为入境。
  • 往来机场：MU261 / MU262 为马德里，MU751 / MU752 为罗马；其它航班取非温州那一端。
  • 上报口岸：固定为 050-温州机场。
  • 不完整记录：姓名 / 证件号 / 国籍 / 性别 / 生日 任一缺失的行将被跳过，并在结果中列出，供你补录。
  • 输出：生成「航班订票数据上传模板」格式 .xlsx（标题行 + 15 列表头 + 旅客数据行），可直接下载。

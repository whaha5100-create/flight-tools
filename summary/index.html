航班运量汇总

输入东航 / 上航 / 中联航的旅客、行李与客座率，一键汇总东上航、联航与总计

✈️
📖 计算规则说明 点击展开
▾
  • 东上航 = 东航 + 上航（逐字段相加）。
  • 联航 = 中联航（直接取中联航数据）。
  • 总计 = 东上航 + 联航 = 东航 + 上航 + 中联航。
  • 出港客座率：东上航 = (东航客座率 + 上航客座率) ÷ 2；联航 = 中联航客座率；总计 = (东上航客座率 + 联航客座率) ÷ 2。结果均带 % 单位。
  • 输入框表达式：支持 + - * / 四则运算（按先乘除后加减），例如 11+65+231 求和得 307、16*78/15 得 83.2；空单元格自动按 0 计算。
  • 所有计算均在浏览器本地完成，数据不上传任何服务器。

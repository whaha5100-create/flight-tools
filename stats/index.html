航班数据统计表格生成系统

上传航班 Excel 模板，系统自动按航司分类统计架次、特服、调时与取消航班，并生成可下载的统计表格。全程在本地浏览器处理，数据不上传服务器。

✈️
📋 统计规则说明 点击展开查看
▼
系统读取 Excel 中的 机号 / 进港航班 / 出港航班 / 前站 / 到站 / 特服 / 备注 等列，按下列规则进行统计与分类。
💡 统计细则
• 航司分类：航班号以 MU / FM / CES 开头计为东上航司，以 KN 开头计为中联航，其余归入「其他」不计入。
• 国内 / 国际：进港的「前站」或出港的「到站」为 马德里 / 罗马 时记为国际，否则为国内。
• 特服统计（仅出港）：取特服列中 | 之后的内容，轮N=轮椅数、童N=无陪儿童数。
• 出港调时：备注列含「调时」字样时，记录对应出港航班号。
• 计划性取消：机号列为 CNL / DEL 判定为取消航班，不计入架次统计。

# 航班工作工具集（GitHub Pages）

三个纯前端工具，部署在 GitHub Pages，免费且永久在线。所有计算均在浏览器本地完成，Excel 不会上传到任何服务器。

## 工具

| 工具 | 路径 | 说明 |
|---|---|---|
| 国际名单转化 | `/manifest/` | 航班名单 → 海关上传模板 |
| 升舱金额计算器 | `/upgrade/` | 按岗位比例分配升舱金额 |
| 航班数据统计 | `/stats/` | 航班 Excel 分类统计 |

## 本地预览

直接用浏览器打开 `index.html` 即可（各子目录自包含，断网也能用）。

## 部署

由 WorkBuddy 通过 `git` 推送到本仓库并启用 GitHub Pages（分支 `main`，根目录）。

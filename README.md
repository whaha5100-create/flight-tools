## 出港排班表（重点功能）

- 上传 Excel，自动筛选出港航班（航班号自动清理 `#`、`&`）
- 计离 **0830 之前** 的航班自动剔除（属于前一组）
- 按计离自动分三段，分页切换查看：
  - 🌅 早上 `0830–1400`
  - ☀️ 下午 `1400–1830`
  - 🌙 晚上 `1830` 以后
- 备注列自动组合：**预推时间 → 特服（出港） → 航班调时**
- 负责栏多选（预设 5 人 + 自定义），导出图片用鲜艳实色标签直观区分
- 每行可「插入 / 删除」；插入的空行可直接编辑
- 登机口默认预填 `2`，只需补后两位
- 草稿：`存草稿` / `读取草稿` / `清空草稿`（存于浏览器 localStorage）
- 导出：当前页 Excel、全部 Excel（早 / 午 / 晚 三段各一表）、导出图片（蓝底白字表头、负责人彩标）
- `一键清空` 整张表（带确认）

## 本地预览

直接用浏览器打开任意目录下的 `index.html` 即可（各子目录自包含，断网也能用）。

## 技术栈

- 纯 HTML / CSS / JS，无后端
- [SheetJS](https://sheetjs.com/) 解析 / 导出 Excel
- html2canvas + jsPDF 生成日报 PDF
- Canvas 绘制导出图片
- localStorage 草稿持久化
